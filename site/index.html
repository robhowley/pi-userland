❯ control plane

Control plane

Session Deck is the signature package.

Pi agents are most useful when they work independently. Session Deck keeps that independence from scattering context across terminal tabs, repos, and worktrees.

pi-session-deck README

See, group, open, return, and launch Pi agents from one deck.

Open Session Deck as a native Pi TUI with /session-deck, or keep it visible as an always-on iTerm2 Toolbelt sidebar.

  • Agents organized by repo. Collapse quiet repos and keep active work in project context.
  • Detailed status at a glance. Branch, worktree, liveness, activity, and PR state stay together.
  • A reliable way back. Focus an iTerm2 session or reattach to the existing tmux session.
  • New isolated agents on demand. Launch Pi on a generated Git worktree in detached tmux.

Bash safety guardrails for destructive commands.

Block catastrophic commands, ask on destructive ones, and allow the rest so agent work can move fast without filesystem roulette.
